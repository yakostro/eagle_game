===============================================
       THE LAST EAGLE - SOARING ADVENTURE
===============================================

Thank you for downloading The Last Eagle!

SYSTEM REQUIREMENTS:
- Windows 10 or later (64-bit)
- 2 GB RAM
- DirectX 11 compatible graphics card
- 500 MB available disk space

HOW TO PLAY:
1. Extract all files from the ZIP to a folder
2. Double-click "The_Last_Eagle.exe" to start the game
3. Enjoy soaring through the skies as The Last Eagle!

CONTROLS:
- W or Up Arrow: Move Up
- S or Down Arrow: Move Down  
- E: Eat Fish
- Space: Drop Fish at Nest
- H: Screech

GAMEPLAY:
As The Last Eagle, guide yourself through the wilderness, catch fish, and bring them back to your nest to feed your family. Avoid obstacles and manage your energy as you soar through different environments in this survival adventure.

TROUBLESHOOTING:
- If the game doesn't start, make sure you've extracted ALL files from the ZIP
- If you get a Windows security warning, click "More info" then "Run anyway"
- For best performance, close other applications while playing
